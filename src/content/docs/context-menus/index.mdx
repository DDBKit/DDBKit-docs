**Context menus** are lists of actions that your bot can add to channels or users. They are accessed
by right clicking a message or a user under Apps. If you need more input from the user rather than just
a user or message, you can respond with modals to the context menu interaction.

```swift
Context("Reverse search image", kind: .message) { i in 
	try await i.respond {
		Message {
			MessageEmbed {
				Title("Google result")
				Image(.exact("https://placehold.co/600x400/EEE/31343C"))
			}.setColor(.red)
		}
	}
}
.isUsableInDMS(true)
.integrationType(.all, contexts: .all)
```
